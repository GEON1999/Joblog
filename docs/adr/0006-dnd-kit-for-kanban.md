# 칸반 드래그앤드롭은 @dnd-kit

칸반 카드 이동에 @dnd-kit/core를 사용한다. headless라 Tailwind 기반 UI에 스타일 제약 없이 결합되고, 포인터·터치·키보드 센서를 모두 지원해 모바일 사용과 접근성 요구를 충족하며, 현재 React DnD 생태계에서 가장 활발히 유지보수되는 선택지다.

## Considered Options

- **@hello-pangea/dnd** — 칸반형 고수준 API라 조립은 가장 빠르지만, 단종된 react-beautiful-dnd의 포크라는 계보와 낮은 커스터마이징 유연성이 부담.
- **pragmatic-drag-and-drop** — 성능은 최고 수준이나 저수준 API라 조립 비용이 크고, 이 규모의 보드에는 과잉.
- **네이티브 HTML5 DnD** — 의존성이 없지만 모바일 터치 미지원과 드래그 프리뷰 제약으로 폰에서도 쓰는 이 앱의 요구와 맞지 않는다.

## Consequences

- 컬럼 정렬(카드 순서 재배치)은 도입하지 않는다 — 컬럼 내 순서는 단계 진입 시각 정렬로 유지되며, 필요해지면 @dnd-kit/sortable을 추가한다.
- 보드는 클라이언트 컴포넌트가 되고, 이동은 낙관적 업데이트 후 서버 액션으로 확정한다.
